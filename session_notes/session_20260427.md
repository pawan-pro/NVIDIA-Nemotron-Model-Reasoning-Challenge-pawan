Interpretation:

- this suggests prompting alone is unlikely to bridge the gap toward the leaderboard range

## What was learned
### 1. Offline evaluation is now feasible
A reliable Kaggle-safe offline Nemotron prompt-eval loop now exists on RTX Pro 6000.

### 2. Prompt-only uplift appears limited
The best result among tested prompts was `0.15` on the 20-row sampled local eval set.
This is useful for relative comparison, but far below the known public baseline and far from leaderboard-contending performance.

### 3. Compute cost is manageable with the reduced staged design
Compared with earlier long-running prompt sweeps, the 20-row eval was much more practical.
The reduced design should be the default for future local prompt checks.

### 4. Hardware selection matters
- P100 path is not viable for this setup
- RTX Pro 6000 is viable
